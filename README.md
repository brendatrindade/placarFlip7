# Placar Flip7

Aplicação web para controle de pontuação do jogo **Flip 7**, desenvolvida com **React + Vite** e publicada via **GitHub Pages**.

**Acesse online:**  [Placar Flip7](https://brendatrindade.github.io/placarFlip7/)

---

## Sobre o Projeto

O **Placar Flip7** é um placar digital simples e intuitivo para acompanhar partidas de Flip 7, facilitando o controle de pontos durante o jogo.

O projeto foi desenvolvido com foco em:

- Interface limpa e responsiva  
- Experiência rápida e fluida  
- Organização de código   

---

## Tecnologias Utilizadas

- **React**
- **Vite**
- **TypeScript** 
- **CSS** 

---

## Funcionalidades

- Cadastrar jogadores
- Gerenciar pontuações
- Registrar rodadas
- Definir vencedor
- Resetar partida
